## Aesthetic Design Philosophy

When implementing UI, the Frontend Dev Agent follows design principles from the `frontend-design` skill to create distinctive, memorable interfaces that avoid generic AI aesthetics.

### Design Thinking Process

Before implementing any UI:

1. **Purpose**: Understand what problem the interface solves and who uses it
2. **Tone**: Commit to a bold aesthetic direction (minimal, maximalist, brutalist, refined, playful, etc.)
3. **Differentiation**: Identify what makes this interface UNFORGETTABLE

### Typography Standards

| Do | Don't |
|----|-------|
| Pair distinctive display fonts with refined body fonts | Default to Inter, Roboto, Arial, system fonts |
| Choose characterful, unexpected font choices | Use overused fonts like Space Grotesk repeatedly |
| Match font personality to the aesthetic vision | Pick generic "safe" choices |

### Color & Theme

- Commit to a cohesive aesthetic with CSS variables
- Use dominant colors with sharp accents
- Avoid timid, evenly-distributed palettes
- **Never**: Purple gradients on white backgrounds

### Motion & Animation

- Focus on high-impact moments: one orchestrated page load > scattered micro-interactions
- Use `animation-delay` for staggered reveals
- Create scroll-triggering and hover states that surprise
- Prioritize CSS-only solutions for HTML; Motion library for React

### Spatial Composition

- Embrace unexpected layouts: asymmetry, overlap, diagonal flow
- Grid-breaking elements when appropriate
- Either generous negative space OR controlled density—not middle-ground

### Backgrounds & Visual Details

- Create atmosphere and depth (not solid colors)
- Use: gradient meshes, noise textures, geometric patterns, layered transparencies
- Add: dramatic shadows, decorative borders, grain overlays
- Match effects to the overall aesthetic

### Anti-Patterns to Avoid

| Never Use | Reason |
|-----------|--------|
| Inter, Roboto, Arial | Overused, generic |
| Purple gradients on white | AI cliché |
| Predictable layouts | Lacks character |
| Cookie-cutter component patterns | No differentiation |
| Space Grotesk (repeatedly) | Converges across generations |
