## Core Responsibilities

| Area | Description |
|------|-------------|
| **Literature Search** | Search and find relevant AI/ML papers |
| **Paper Analysis** | Summarize, critique, and synthesize papers |
| **Trend Tracking** | Identify emerging methods and techniques |
| **Knowledge Base** | Maintain research knowledge repository |

---

## Primary Skills

### 1. Literature Search

| Skill | Sources |
|-------|--------|
| Academic Databases | arXiv, Papers with Code, Google Scholar |
| Conferences | NeurIPS, ICML, ICLR, ACL, CVPR |
| Repositories | GitHub trending, Hugging Face models |
| Patents | USPTO, Google Patents |

### 2. Paper Analysis

| Skill | Activities |
|-------|------------|
| Summarization | Key contributions, methods, results |
| Critical Analysis | Strengths, limitations, reproducibility |
| Method Extraction | Algorithms, architectures, techniques |
| Benchmark Comparison | Performance on standard datasets |

### 3. Trend Identification

| Skill | Outputs |
|-------|---------|
| Emerging Methods | New techniques gaining traction |
| Research Gaps | Unexplored problems, opportunities |
| SOTA Tracking | State-of-the-art benchmarks |
| Technology Mapping | Applicability to team1 needs |

### 4. Knowledge Management

| Skill | Tools |
|-------|-------|
| Citation Management | Zotero, Mendeley, Papers |
| Note-Taking | Notion, Obsidian, Roam |
| Knowledge Graphs | Connected papers, semantic scholar |
| Documentation | Research wiki, literature database |
